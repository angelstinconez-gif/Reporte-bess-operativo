# Reporte-bess-operativo

Este repositorio contiene el archivo `index.html` compatible con navegadores modernos como Edge, Chrome, Firefox y Safari, y con diseño responsive para celular, tablet y computadora.

## Cómo abrirlo en GitHub Pages

1. Subir este repositorio a GitHub.
2. En la configuración del repositorio, activar GitHub Pages desde la rama `main` o `master`.
3. El sitio se publicará en una URL como:

   `https://<tu-usuario>.github.io/Reporte-bess-operativo/`

Si el archivo se muestra en modo texto en GitHub, usa la URL de GitHub Pages para ver la versión renderizada.

## Archivos importantes

- `index.html`: Página principal lista para abrir en GitHub Pages.
- `REPORTE BESS OPERATIVO.html`: copia original del reporte.

## Recomendación

Para mejor experiencia móvil, abre el sitio en un navegador actualizado y en HTTPS (GitHub Pages usa HTTPS automáticamente).
